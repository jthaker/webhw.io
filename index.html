Summary: Global Weather Plots



Visualizations



This project was an analysis on weather fluctuation as one nears the equator. The OpenWeatherMap API was used to assemble a dataset of over 500 cities.


Plots of various variables of weather vs. latitude were created using matplotlib. Variables included: temperature, cloudiness, wind-speed and humidity.
